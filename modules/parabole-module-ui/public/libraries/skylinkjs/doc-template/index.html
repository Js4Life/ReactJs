1. Reference our Libraries from our hosted CDN

To begin, you may start by including this script from our hosted cdn libraries. The compressed versions are production ready codes which improves page speed and load. It's recommended to use the compressed versions for simply using it and the uncompressed versions for debugging the library.

The complete version consists of the Socket.io client library and Temasys version of AdapterJS and the Skylink library. The standalone version consists of only the Skylink library which require reference the libraries by your own.

You may replace the word latest with the version of Skylink you wish to use.

Dependencies
  • Socket.io (Client)
  • AdapterJS
  • Download SkylinkJS

        <!-- Completed version uncompressed -->
        <script src="//cdn.temasys.com.sg/skylink/skylinkjs/latest/skylink.complete.js"></script>
        <!-- Standalone version uncompressed -->
        <script src="//cdn.temasys.com.sg/skylink/skylinkjs/latest/skylink.debug.js"></script>
        <!-- Completed version compressed -->
        <script src="//cdn.temasys.com.sg/skylink/skylinkjs/latest/skylink.complete.min.js"></script>
        <!-- Standalone version compressed -->
        <script src="//cdn.temasys.com.sg/skylink/skylinkjs/latest/skylink.debug.min.js"></script>

3. Integrating Skylink into your Web Application

Here's how you can integrate Skylink into your Web Application with simple basic video conferencing functionalities. 

        <!doctype html>
        <html>
          <head>
            <title>Simple Skylink Application</title>
            <script src="//cdn.temasys.com.sg/skylink/skylinkjs/latest/skylink.complete.js"></script>
          </head>
          <body>
            <video id="me" autoplay muted></video>
            <video id="other" autoplay></video>
            <script>
              window.onload = function () {
                var meElement = document.getElementById('me');
                var otherElement = document.getElementById('other');

                Skylink.on('incomingStream', function (stream, peerId, peerInfo, isSelf) {
                  if (isSelf) {
                    stream.attach(meElement);
                  } else {
                    stream.attach(otherElement);
                  }
                });

                // Optional for debugging
                // Skylink.setLogLevel(Skylink.LOG_LEVEL.DEBUG);

                Skylink.init(apiKey, function () {
                  Skylink.joinRoom({
                    audio: true,
                    video: true
                  });
                });
              };
            </script>
          </body>
        </html>
